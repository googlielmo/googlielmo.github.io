Function bd_windows_exec_token

1

func bd_windows_exec_token(arg: string) -> string

Normalize one Windows executable token for cmd.exe .

Relative ./… command paths are valid under POSIX shells but cmd.exe interprets the leading ./ as a command named . . Rewriting the slash separators to .... keeps repo-local temp executables runnable when the temp-root fallback is the current directory.

Parameters:

  • arg: Raw executable token.

Returns: Command token with cmd.exe-safe relative separators.
