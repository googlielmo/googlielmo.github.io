Function type_new

1

func type_new(kind: TypeKind, module_name: string, name: string, inner: Type*?, params: VectorBase*?, result: Type*?) -> Type*

Create a type descriptor from raw components.

Parameters:

  • kind: Type kind tag.
  • module_name: Defining module name for nominal types.
  • name: Type name for builtin and nominal kinds.
  • inner: Owned inner type for pointer and nullable kinds.
  • params: Owned parameter vector for function kinds.
  • result: Owned result type for function kinds.

Returns: New type descriptor.
